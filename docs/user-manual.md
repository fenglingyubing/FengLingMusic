## 1. 简介

### 1.1 关于风铃音乐

风铃音乐是一款现代化的跨平台音乐播放器，支持 Windows 和 Android 双平台。它具备强大的本地音乐管理功能、在线音乐搜索播放、高清歌词显示等特性，并提供 120fps 超流畅的动画体验。

### 1.2 主要特性

- **本地音乐管理**: 自动扫描并管理本地音乐库
- **在线音乐**: 搜索并播放来自多个平台的在线音乐
- **高品质音频**: 支持多种音频格式和高音质播放
- **歌词显示**: 实时同步显示滚动歌词，支持翻译
- **播放列表**: 创建和管理个性化播放列表
- **高刷新率**: 支持 120/144/165Hz 显示器，动画流畅无卡顿
- **跨平台**: Windows 和 Android 无缝体验

### 1.3 系统要求

#### Windows
- 操作系统: Windows 10 (1809+) 或 Windows 11
- 处理器: Intel Core i3 或同等 AMD 处理器
- 内存: 4GB RAM (推荐 8GB)
- 硬盘空间: 200MB 安装空间 + 音乐文件存储空间
- 显示器: 1280x720 或更高分辨率（支持 120Hz+ 高刷新率）

#### Android
- 操作系统: Android 8.0 (API 26) 或更高版本
- 处理器: 双核 1.5GHz 或更高
- 内存: 2GB RAM (推荐 4GB)
- 存储空间: 100MB 安装空间 + 音乐文件存储空间

## 2. 安装说明

### 2.1 Windows 安装

1. 从 [GitHub Releases](https://github.com/fenglingyubing/FengLingMusic/releases) 下载最新版本的安装包 (`FengLingMusic-Setup-x64.exe`)
2. 双击安装包启动安装向导
3. 按照向导提示完成安装
4. 安装完成后，从开始菜单或桌面快捷方式启动应用

#### 便携版安装（可选）
- 下载便携版压缩包 (`FengLingMusic-Portable-x64.zip`)
- 解压到任意目录
- 运行 `FengLingMusic.exe` 启动应用

### 2.2 Android 安装

#### 方式一：Google Play（推荐）
1. 打开 Google Play 商店
2. 搜索"风铃音乐"或"FengLing Music"
3. 点击"安装"按钮
4. 安装完成后打开应用

#### 方式二：APK 安装
1. 从 [GitHub Releases](https://github.com/fenglingyubing/FengLingMusic/releases) 下载最新版本的 APK 文件
2. 在手机设置中允许"从未知来源安装应用"
3. 使用文件管理器打开 APK 文件
4. 按照提示完成安装

### 2.3 首次启动

首次启动应用时，系统会请求必要的权限：

**Windows**:
- 文件访问权限（用于扫描本地音乐）
- 网络访问权限（用于在线功能）

**Android**:
- 存储权限（读取音乐文件）
- 通知权限（显示播放控制）
- 可选：耳机线控权限（支持耳机按钮控制）

## 3. 快速开始

### 3.1 添加音乐

#### 自动扫描
1. 进入"设置" > "扫描路径"
2. 点击"添加路径"选择音乐文件夹
3. 点击"开始扫描"，应用会自动扫描并添加音乐

#### 手动添加
- 将音乐文件复制到扫描路径中
- 返回应用，点击刷新按钮更新音乐库

### 3.2 播放音乐

1. 在"音乐库"页面浏览您的音乐
2. 点击任意歌曲开始播放
3. 使用底部播放控制栏控制播放

### 3.3 创建播放列表

1. 进入"播放列表"页面
2. 点击右上角"+"按钮
3. 输入播放列表名称
4. 长按歌曲，选择"添加到播放列表"

## 4. 功能介绍

### 4.1 音乐库管理

#### 4.1.1 浏览音乐
- **歌曲列表**: 查看所有歌曲，支持搜索和排序
- **艺术家视图**: 按艺术家分类浏览
- **专辑视图**: 按专辑分类浏览，网格布局展示封面
- **文件夹视图**: 按文件夹结构浏览

#### 4.1.2 搜索功能
- 在搜索框中输入关键词
- 支持搜索歌曲标题、艺术家、专辑
- 实时显示搜索结果
- 查看搜索历史

#### 4.1.3 排序和筛选
- 按标题、艺术家、专辑、时长排序
- 升序/降序切换
- 按格式、音质筛选

### 4.2 播放控制

#### 4.2.1 基础控制
- **播放/暂停**: 点击播放按钮
- **上一曲/下一曲**: 点击跳转按钮
- **进度调整**: 拖动进度条
- **音量调整**: 点击音量图标，拖动音量滑块

#### 4.2.2 播放模式
- **顺序播放**: 按列表顺序播放
- **随机播放**: 随机选择歌曲播放
- **单曲循环**: 重复播放当前歌曲
- **列表循环**: 播放完列表后重新开始

#### 4.2.3 播放队列
1. 点击底部播放栏展开全屏播放器
2. 点击"播放队列"按钮查看队列
3. 拖拽歌曲调整播放顺序
4. 向左滑动删除歌曲
5. 点击"清空队列"清除所有歌曲

### 4.3 播放列表

#### 4.3.1 创建播放列表
1. 进入"播放列表"页面
2. 点击"新建播放列表"
3. 输入名称和描述（可选）
4. 添加歌曲到播放列表

#### 4.3.2 编辑播放列表
- **添加歌曲**: 长按歌曲 > "添加到播放列表"
- **删除歌曲**: 在播放列表详情页向左滑动歌曲
- **调整顺序**: 长按拖动歌曲
- **批量操作**: 点击"编辑"按钮进入批量模式

#### 4.3.3 导入/导出
- **导出**: 播放列表 > 菜单 > "导出为 M3U"
- **导入**: 播放列表 > 菜单 > "导入播放列表" > 选择 M3U/M3U8 文件

### 4.4 在线音乐

#### 4.4.1 搜索在线音乐
1. 进入"在线音乐"页面
2. 输入搜索关键词
3. 选择音乐平台（网易云、QQ音乐、酷狗音乐）
4. 浏览搜索结果

#### 4.4.2 播放在线音乐
- 点击歌曲直接播放
- 添加到播放队列
- 在线音乐会自动缓存，下次播放更快

#### 4.4.3 下载音乐
1. 长按歌曲 > "下载"
2. 选择音质（标准/高品质/无损）
3. 进入"下载管理"查看下载进度
4. 下载完成后自动添加到本地库

### 4.5 歌词功能

#### 4.5.1 查看歌词
- 播放时自动加载歌词（在线歌曲）
- 本地歌曲支持读取同名 .lrc 文件
- 点击底部播放栏展开全屏播放器查看歌词

#### 4.5.2 歌词交互
- **自动滚动**: 歌词随播放进度自动滚动
- **手动滚动**: 手指滑动浏览全部歌词
- **跳转播放**: 点击歌词行跳转到对应时间
- **翻译显示**: 如有翻译，会显示在原文下方

#### 4.5.3 桌面歌词（Windows）
1. 进入"设置" > "歌词设置"
2. 开启"显示桌面歌词"
3. 桌面歌词窗口会悬浮在其他窗口之上
4. 拖动歌词窗口调整位置
5. 右键歌词窗口调整样式

### 4.6 收藏和历史

#### 4.6.1 收藏歌曲
- 点击歌曲的心形图标收藏
- 进入"我喜欢"查看所有收藏的歌曲
- 再次点击取消收藏

#### 4.6.2 播放历史
- **最近播放**: 按时间倒序显示播放记录
- **最常播放**: 按播放次数排序显示热门歌曲
- 系统会自动记录播放历史和统计数据

### 4.7 主题和外观

#### 4.7.1 切换主题
1. 进入"设置" > "外观"
2. 选择主题模式：
   - 浅色主题
   - 深色主题
   - 跟随系统

#### 4.7.2 自定义主题色
1. 进入"设置" > "外观" > "主题色"
2. 选择预设颜色或自定义颜色
3. 实时预览效果
4. 点击"应用"保存

#### 4.7.3 动画设置
- **高刷新率**: 开启 120fps 动画（需要支持的显示器）
- **目标帧率**: 选择 60/90/120 fps
- **动画速度**: 调整动画速度（慢速/正常/快速）

### 4.8 设置选项

#### 4.8.1 播放设置
- **淡入淡出**: 切歌时音频淡入淡出时长（0-3秒）
- **音量正常化**: 自动调整不同歌曲的音量
- **默认播放模式**: 启动时的播放模式

#### 4.8.2 扫描设置
- **扫描路径**: 添加/删除音乐文件夹
- **扫描格式**: 选择要扫描的音频格式
- **自动扫描**: 应用启动时自动扫描

#### 4.8.3 缓存管理
- **查看缓存大小**: 显示当前缓存占用空间
- **清理缓存**: 清除播放缓存和临时文件
- **清理封面缓存**: 清除专辑封面缓存

#### 4.8.4 数据备份
- **备份数据**: 导出播放列表、设置、历史记录
- **恢复数据**: 从备份文件恢复数据

## 5. 常见问题

### Q1: 应用无法扫描到我的音乐文件？
**A**:
1. 确认音乐文件格式是否支持（MP3, FLAC, WAV, AAC, OGG, M4A）
2. 检查扫描路径是否正确添加
3. 确认应用具有读取该文件夹的权限
4. 尝试手动触发"重新扫描"

### Q2: 播放在线音乐时提示"无法获取播放地址"？
**A**:
1. 检查网络连接是否正常
2. 该歌曲可能存在版权限制
3. 尝试切换其他音乐平台搜索
4. 更新应用到最新版本

### Q3: 歌词不同步或不显示？
**A**:
- 在线歌曲：系统会自动获取歌词，如果没有显示可能是歌词源没有该歌曲的歌词
- 本地歌曲：需要在音乐文件同目录下放置同名的 .lrc 歌词文件
- 检查歌词文件格式是否正确（LRC 格式）

### Q4: Windows 系统托盘图标不显示？
**A**:
1. 检查系统托盘设置，确保允许显示该应用图标
2. 重启应用
3. 在"设置" > "高级"中开启"显示系统托盘图标"

### Q5: Android 后台播放被系统杀死？
**A**:
1. 在系统设置中允许应用后台运行
2. 关闭省电模式或将应用添加到白名单
3. 锁定应用在最近任务中
4. 部分手机需要在厂商的手机管家中设置自启动

### Q6: 120fps 动画不流畅？
**A**:
1. 确认显示器支持高刷新率（120Hz+）
2. 在"设置" > "外观" > "动画设置"中检查是否开启高刷新率
3. 关闭其他占用资源的应用
4. 降低动画质量或目标帧率

### Q7: 下载的音乐找不到？
**A**:
- 下载完成后会自动添加到本地音乐库
- 在"音乐库"中搜索歌曲名
- 文件保存在 "文档/FengLingMusic/Downloads" 目录

### Q8: 如何卸载应用？
**A**:
- **Windows**: 控制面板 > 程序和功能 > 找到"风铃音乐"并卸载
- **Android**: 长按应用图标 > 卸载，或在系统设置中卸载

## 7. 故障排除

### 7.1 应用崩溃或无法启动

**可能原因**:
- 系统文件损坏
- 权限不足
- 配置文件错误

**解决方案**:
1. 重启应用
2. 重启设备
3. 重新安装应用
4. 清除应用数据（注意：会丢失设置）

### 7.2 音频播放问题

**可能原因**:
- 音频文件损坏
- 不支持的格式
- 音频驱动问题

**解决方案**:
1. 尝试播放其他歌曲
2. 检查音频文件是否损坏
3. 更新音频驱动（Windows）
4. 在其他播放器中测试该文件

### 7.3 网络连接问题

**可能原因**:
- 无网络连接
- 防火墙阻止
- 代理设置不当

**解决方案**:
1. 检查网络连接
2. 在防火墙中允许应用联网
3. 检查代理设置

### 7.4 性能问题

**症状**: 界面卡顿、动画掉帧

**解决方案**:
1. 关闭不必要的后台应用
2. 降低动画质量
3. 减少音乐库大小（删除不需要的歌曲）
4. 清理缓存

### 7.5 获取日志文件

如需提交问题报告，请提供日志文件：

**Windows**:
```
%APPDATA%/FengLingMusic/logs/
```

**Android**:
设置 > 关于 > 导出日志
